# 浮生图片混淆 (FIXpictures)

一个轻量级的浏览器端图片混淆/解混淆工具，支持刮刮乐展示效果和液态玻璃视觉特效。

## ✨ 核心功能

### 图片处理
- **🔒 图片混淆**：安全的图片内容加密保护
- **🔓 图片解混淆**：快速恢复原始图片
- **🎫 刮刮乐效果**：交互式解密展示（直接显示/刮刮乐/半透明刮刮乐）

### 视觉特效
- **💎 液态玻璃效果**：可自定义的玻璃质感视觉效果
- **🌙 主题切换**：深色/浅色主题无缝切换
- **🎨 动态背景**：自动生成的渐变动态背景

### 用户体验
- ** 响应式设计**：完美适配桌面和移动设备
- **🖱️ 多种交互方式**：文件选择、拖放、剪切板粘贴
- **⚡ 自动处理**：预设自动混淆/解混淆模式

## 🚀 快速使用

### 本地运行
1. 下载项目文件
2. 用浏览器打开 `index.html`
3. 开始使用图片混淆功能

### 在线部署
支持 Cloudflare Pages 部署，详见 [DEPLOYMENT.md](./DEPLOYMENT.md)

## 📖 功能说明

### 基本操作
- **选择图片**：点击 `📁 选择图片` 或拖放图片到页面
- **粘贴图片**：支持 `Ctrl+V` 或点击 `📋 粘贴图片`
- **自动处理**：预设图片加载后的自动操作模式
- **显示模式**：选择解密时的展示方式

### 控制按钮
- `🔒 混淆`：加密当前图片
- `🔓 解混淆`：解密已加密图片
- `↻ 还原`：恢复到原始状态

### 高级设置
- **液态玻璃效果**：自定义模糊、透明度、色调等参数
- **主题切换**：右上角按钮切换深色/浅色模式

## 📄 项目信息

### 技术架构
- **前端**：HTML5 + CSS3 + JavaScript
- **加密**：Web Crypto API
- **图片处理**：Canvas API + EXIF.js
