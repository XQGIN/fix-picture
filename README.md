# 浮生图片混淆 (FIXpictures)

这是一个浏览器端的图片混淆/解混淆并带有"刮刮乐"展示效果的轻量网页工具。支持液态玻璃效果、动态背景、主题切换和全面的防爬虫保护。

## ✨ 项目特色

- **🔒 图片混淆加密**：安全的图片内容混淆和解混淆功能
- **🎫 刮刮乐效果**：交互式解密展示，支持直接显示、刮刮乐和半透明刮刮乐模式
- **💎 液态玻璃效果**：可自定义的玻璃质感视觉效果
- **🌙 主题切换**：深色/浅色主题无缝切换
- **🎨 动态背景**：渐变动态背景增强视觉体验
- **🛡️ 防爬虫保护**：多层防护机制防止自动化访问
- **📱 响应式设计**：完美适配桌面和移动设备

## 🚀 快速开始

### 本地使用
1. 下载项目文件到本地
2. 用浏览器打开 `index.html` 文件
3. 开始使用图片混淆功能

### 在线部署
项目支持通过 Cloudflare Pages 部署，提供更好的性能和安全性。详见 [DEPLOYMENT.md](./DEPLOYMENT.md)。

## 📖 功能说明

### 主要交互
- **选择图片**：点击 `📁 选择图片` 打开文件对话框，选择本地图片
- **拖放图片**：将图片拖入页面中间区域
- **粘贴图片**：在支持的浏览器中复制图片后，按 `Ctrl+V` 或点击 `📋 粘贴图片`
- **自动处理设置**：选择页面在加载图片后自动执行的操作：不处理 / 自动混淆 / 自动解混淆
- **解密显示模式**：选择解密显示时的展示方式：直接显示 / 刮刮乐 / 半透明刮刮乐

### 按钮功能
- `🔒 混淆`：对当前图片执行混淆操作
- `🔓 解混淆`：对已混淆的图片执行解混淆
- `↻ 还原`：将当前视图和图片恢复到加载前的原始状态

### 高级功能
- **💎 液态玻璃效果**：点击玻璃效果按钮自定义模糊强度、透明度、亮度等参数
- **🌙 主题切换**：右上角按钮切换深色/浅色主题
- **🎨 动态背景**：页面加载时自动生成渐变动态背景

## 🛡️ 安全特性

### 防爬虫机制
项目集成了多层防爬虫保护：
- **HTML Meta标签防护**：禁止搜索引擎索引和爬取
- **robots.txt配置**：明确禁止所有爬虫访问
- **JavaScript检测**：实时检测人类行为和爬虫特征
- **Cloudflare Worker防护**：服务器端爬虫识别和拦截
- **安全头配置**：防止点击劫持和MIME类型嗅探

### 隐私保护
- 所有图片处理均在浏览器本地完成
- 不收集任何用户数据或图片内容
- 支持离线使用

## 🔧 故障排查

### 常见问题
- **图片加载失败**：确认文件格式支持（JPEG/PNG/WebP）
- **粘贴功能无效**：检查浏览器剪贴板权限
- **功能异常**：尝试刷新页面或使用推荐浏览器
- **防爬虫误报**：如被误判为爬虫，请使用正常浏览器访问

### 技术支持
如遇问题，请检查：
1. 浏览器控制台是否有错误信息
2. 网络连接是否正常
3. 文件权限是否正确

## 📄 项目信息

### 技术栈
- **前端**：纯 HTML5 + CSS3 + JavaScript
- **加密**：Web Crypto API
- **图片处理**：Canvas API + EXIF.js
- **部署**：Cloudflare Pages + Workers
